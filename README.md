# 🧠 ADALINE — Red Neuronal Artificial

Implementación de **ADALINE** (Adaptive Linear Neuron) desarrollada en MATLAB como laboratorio de Inteligencia Artificial. Incluye entrenamiento con múltiples casos de prueba y visualización de resultados.

![MATLAB](https://img.shields.io/badge/MATLAB-App_Designer-0076A8?style=flat&logo=mathworks&logoColor=white)
![IA](https://img.shields.io/badge/Red_Neuronal-ADALINE-6A5ACD?style=flat)

---

## 📌 Descripción

ADALINE es una red neuronal de una sola capa que usa la **regla de aprendizaje de Widrow-Hoff** (mínimos cuadrados) para ajustar sus pesos. A diferencia del Perceptrón, ADALINE minimiza el error cuadrático medio durante el entrenamiento, lo que la hace más precisa en problemas de clasificación lineal.

---

## ✨ Casos de prueba

| Archivo | Descripción |
|---|---|
| `entradas_caso1.txt` | Conjunto de entradas — caso 1 |
| `entradas_caso2.txt` | Conjunto de entradas — caso 2 |
| `entradas_caso3.txt` | Conjunto de entradas — caso 3 |
| `pesos.txt` | Pesos iniciales de la red |
| `resultados_entrenamiento.txt` | Resultados después del entrenamiento |

---

## 🔬 Diferencia entre Perceptrón y ADALINE

| Característica | Perceptrón | ADALINE |
|---|---|---|
| Función de activación | Escalón | Lineal |
| Regla de aprendizaje | Perceptrón | Widrow-Hoff |
| Minimiza | Errores de clasificación | Error cuadrático medio |
